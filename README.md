# 🌟 Overview

**InkFlow** bridges the gap between digital convenience and the human touch.  
By combining **neural handwriting synthesis** with **advanced AI note extraction**, InkFlow lets you create print-ready handwritten documents from raw text, PDFs, or Word files.

> ✨ **The experience** – earthy cream palettes, rich brown accents, soft layered shadows – feels like writing on high‑quality stationery, but with the power of AI at your fingertips.

---

<br>

# 🚀 Key Features

| Category | What you get |
|----------|---------------|
| **✍️ Handwriting Synthesis** | Human‑like slant, rotation & spacing variations <br/> 10+ authentic styles (elegant cursive to neat print) <br/> Adjust font size, spacing, line height & ink color (blue, black, red, green) |
| **🧠 AI-Powered Studio** | 📚 **AI Notes Generator** – condense textbooks/transcripts into structured study notes <br/> 📝 **AI Quiz Generator** – turn any document into a practice quiz <br/> 📄 **Document Intelligence** – upload PDF, DOCX, or TXT files |
| **🖨️ Premium Preview & Export** | Realistic paper backgrounds: ruled, grid, vintage, parchment, recycled <br/> High‑res **multi‑page PDF** or **crystal‑clear PNG** sets <br/> Sticky header + tabbed navigation – clean two‑column workspace |

# 🖥️ Tech Stack

</div>

- **Frontend:** ⚛️React 18 with Vite
- **Styling:** 🎨Tailwind CSS
- **Animations:** Motion (🎞️Framer Motion)
- **AI Integration:** ✨Google Gemini API (via server-side proxy)
- **Icons:** 🤖Lucide React
- **Export:** 🖼️html-to-image & jsPDF

<div align="center">

## Installation

</div>

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/inkflow.git
cd inkflow
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Configure Environment Variables

Create a `.env` file in the root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

---

### 4️⃣ Run Development Server

```bash
npm run dev
```

---

### 5️⃣ Build For Production

```bash
npm run build
```
